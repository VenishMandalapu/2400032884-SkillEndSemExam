# Student Dashboard

A responsive student dashboard built with HTML, CSS, and JavaScript. This dashboard allows students to view their courses, marks, and notifications in a clean, user-friendly interface.

## Features

- **Responsive Design**: Adapts to both desktop and mobile screens using CSS Grid.
- **Sidebar Navigation**: Easy access to different sections like Dashboard, Courses, Marks, etc.
- **Top Bar**: Displays user information and welcome message.
- **Quick Stats**: Cards showing total courses, average marks, and notification count.
- **Notifications**: Section for displaying important notifications.
- **Recent Activities**: Cards for recent student activities.
- **Mobile Toggle**: Sidebar can be toggled on mobile devices for better usability.

## Technologies Used

- HTML5
- CSS3 (with CSS Grid for layout)
- JavaScript (for interactive elements like sidebar toggle)

## Project Structure

```
student-dashboard/
├── index.html      # Main HTML file
├── styles.css      # CSS styles for layout and responsiveness
├── script.js       # JavaScript for interactivity
└── README.md       # This file
```

## How to Run

1. Clone or download the project files.
2. Open `index.html` in a web browser.
3. The dashboard will load and be fully functional.

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge

## Customization

- Modify `styles.css` to change colors, fonts, or layout.
- Update `index.html` to add more sections or change content.
- Enhance `script.js` for additional interactivity.

## Responsive Breakpoints

- Desktop: > 768px
- Mobile: ≤ 768px

On mobile, the sidebar is hidden by default and can be toggled using the "Menu" button in the top bar.
